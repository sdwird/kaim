Usage

01

Open your executor

Launch a compatible Lua script executor on PC. This script does not run on mobile.

02

Choose a release

Pick Stable for a tested build, or Experimental for cutting-edge features. Both are above.

03

Paste & execute

Paste the loadstring command into your executor and run. Activates automatically.

⚠️

Config resets after major updates

Due to frequent changes in the script's memory structure, old config files can break after major version updates. If settings fail to load, delete your old config and create a new one.
