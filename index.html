Render Your Custom Config

Want to see your exact kaim config in the browser instead of the demo? You can dump your live UI directly to a JSON file using the Obsidian Extractor.

1. Run the Extractor Script

Execute this snippet in your executor while kaim is running.

-- Extractor
loadstring(game:HttpGet("https://raw.githubusercontent.com/mspaint-cc/assets/refs/heads/main/uidata/extractor.lua"))()

2. Update the HTML

The script saves a file named ObsidianExtracted.json in your executor's workspace folder. Place that file in your website directory and update the HTML tag:

<Obsidian
  title="kaim"
  ui-data-src="./ObsidianExtracted.json">
</Obsidian>
